# ☁️ Nebula Forecast 🌈

> App de previsão do tempo em tempo real com interface moderna, glassmorphism e previsão do tempo para 5 dias!

---

## 📖 Sobre o Projeto

> A **Nebula Forecast** é uma aplicação web de previsão do tempo que combina design (que eu escolhi a dedo com muito carinho) com funcionalidade robusta.
> 
> O tema da página é dinâmico, e alterna entre dia e noite, além do glassmorphism e animações suaves.
>
> O app foi desenhado para fornecer informações climáticas em tempo real de qualquer cidade do Brasil e do mundo! 🌏

---

## ✨ Funcionalidades

- 🌤️ Clima em tempo real via Open-Meteo API
- 📅 Previsão para os próximos 5 dias com temperaturas máximas e mínimas
- 🕐 Relógio em tempo real atualizado a cada segundo
- 🌓 Tema dinâmico: alterna automaticamente entre dia e noite
- 🎨 Design com glassmorphism e animações suaves
- 📱 Totalmente responsivo
- 🧪 19 testes automatizados com Jest

## 🧪 Testes

19 testes automatizados cobrindo funções principais, chamadas de API, validação de dados, renderização e casos extremos.

```bash
# Instalar dependências
npm install

# Rodar todos os testes
npm test

# Com coverage
npm run test:coverage
```

---

## 🚀 Como Rodar

```bash
# Clone o repositório
git clone https://github.com/nadiacaricatto/nebula-forecast.git

# Entre na pasta
cd nebula-forecast

# Instale as dependências
npm install

# Abra o index.html no navegador ou use Live Server no VS Code
```
